# 墨色修仙 · 程序员黄历 使用指南

## 📖 项目简介

这是一款融合了传统中式水墨美学与现代程序员文化的趣味应用。通过输入你的生辰信息，获取每日的工作运势指南，同时结合实时BTC行情，为你的coding之路增添一丝玄学色彩。

## ✨ 核心功能

### 1. 运势生成引擎
- 基于生辰八字生成个性化的每日运势
- 程序员特色的"宜/忌"事项（如：宜写单元测试、忌周五上线）
- 代码质量指数（0-100）预测
- 每日玄学预言

### 2. 实时行情集成
- BTC/USDT 实时价格展示
- 24小时涨跌幅统计
- 基于运势的涨跌预测（玄学向）

### 3. 视觉体验
- **水墨背景**：动态Canvas水墨晕染效果
- **卷轴展开**：流畅的Framer Motion动画
- **中式设计**：宣纸质感、印章元素
- **响应式布局**：支持桌面和移动端

### 4. 交互特性
- 鼠标移动触发水墨粒子效果
- 卷轴式渐进展示
- 一键分享运势（支持原生分享API）

## 🎯 使用流程

### 第一步：输入信息
1. 输入道号（可选）- 如"老黄"、"小李"
2. 选择出生年月日（必填）
3. 选择性别
4. 点击"🔮 卜算今日运势"

### 第二步：查看运势
等待水墨晕染动画，卷轴将缓缓展开，展示：
- 今日日期（阳历 + 农历 + 程序员节气）
- 代码质量指数
- 今日宜/忌事项
- 玄学预言（含BTC趋势预测）
- 幸运颜色和幸运编程语言

### 第三步：分享或重新卜算
- 点击"分享运势"按钮分享到社交媒体
- 点击"重新卜算"返回输入页面

## 🔧 技术特性

### 运势算法
- 使用基于日期的伪随机数生成器（seeded random）
- 确保同一天、同一生日的用户获得相同运势
- 每日0点自动刷新运势

### BTC行情
- 当前为模拟数据
- 可替换为真实Binance API（详见 `API_INTEGRATION.md`）
- 支持实时WebSocket更新

### 水墨效果
- 纯Canvas实现，无需额外资源
- 粒子系统模拟水墨扩散
- 性能优化，支持移动设备

## 🎨 设计理念

### 新中式美学
- **宣纸质感**：米黄色背景，营造古朴氛围
- **水墨晕染**：动态墨迹，传统与科技结合
- **印章元素**：右下角红色印章，增加仪式感
- **书法字体**：标题使用楷体风格

### 程序员文化
- **代码化运势**：将传统宜忌转化为编程场景
- **技术栈幸运**：每日推荐幸运编程语言
- **极客幽默**：融入程序员梗（如"水逆期"、"五行利多"）

### 玄学与科技
- **BTC趋势**：将加密货币与命理结合
- **算法玄学**：用代码演绎传统占卜
- **数据驱动**：基于生辰数据的个性化内容
