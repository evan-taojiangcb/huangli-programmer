## ✨ 核心功能

### 🎨 视觉设计
- **新中式美学**：宣纸质感、水墨晕染、印章元素

- **前端框架**：React 18+ with TypeScript
- **样式方案**：Tailwind CSS 4.0（含自定义工具类）
- **动画库**：Motion (Framer Motion)
- **图标库**：Lucide React
- **Canvas绘图**：原生Canvas API（水墨、烟花、红包）

## 🎆 春节特色技术实现

### Canvas多层动画系统
```typescript
// 同时渲染三种效果
- 水墨粒子：InkParticle 类
- 烟花效果：Firework 粒子系统
- 红包飘落：RedEnvelope 动画
```

### CSS动画
```css
/* 金光闪闪效果 */
.shimmer-text {
  background: linear-gradient(90deg, #DC2626, #FBBF24, #DC2626);
  animation: shimmer 3s linear infinite;
}

/* 竖排文字（对联） */
.writing-mode-vertical {
  writing-mode: vertical-rl;
  text-orientation: upright;
}
```

### Motion动画配置
- 灯笼摇摆：y轴 + 旋转动画
- 福字飘动：旋转 + 缩放组合
- 卷轴展开：多层级延迟动画

## 🎲 运势算法

应用使用基于日期的伪随机数生成器（seeded random），确保：
- 同一天、同一生日的用户获得相同运势
- 每日0点自动刷新
- 运势结果可复现
- 春节特色内容随机出现
